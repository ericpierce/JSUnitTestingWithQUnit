Why Unit Test JavaScript?

What makes
JavaScript unit testing hard

  • Inline JS (mixed in HTML tags)
  • JS in markup template file
  • JS tightly coupled with HTML markup
  • JS tightly coupled with backend services

QUnit

  • Created by John Resig (jQuery, etc.)
  • Used by jQuery family of projects for unit testing
  • Can be used to unit test any JS (no jQuery dependency, etc.)
